# Week 3 Power BI Dashboard Build Guide

## Files to load
Load these two files from this folder:

1. week3_dashboard_sales_model.csv
2. week3_inventory_model.csv

## Dashboard page layout

Top cards:
- Total Revenue
- Net Revenue
- Total Orders
- Average Order Value
- Return Rate

Main visuals:
- Line chart: Month_Name on axis, Total Revenue as value
- Bar chart: Channel on axis, Total Revenue as value
- Bar chart: Product_Name on axis, Total Revenue as value, Top 10 filter
- Bar chart or map: City as location/axis, Total Revenue as value
- Bar chart: Day_of_Week on axis, Total Orders as value
- Donut chart: Category as legend, Total Revenue as value
- Table: Product_Name, Category, Total Quantity Sold, Total Revenue, Return Rate

Slicers:
- Year
- Month_Name
- Channel
- City
- Category

Suggested title:
Omnichannel Retail Sales Dashboard

## What to say
In Week 3, I created a Power BI dashboard by combining sales, customer, product, return, and inventory data. The dashboard shows revenue, orders, channel performance, product performance, city-wise sales, return rate, and inventory status using interactive filters.
